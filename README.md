# Rust Calculator

Базовый калькулятор с четырьмя операциями: сложение, вычитание, умножение и деление. Включает обработку ошибок (деление на ноль).

## Запуск

cargo run


## Использование

Измените переменные в `main()`:


let a = 10.0;
let b = 5.0;
let operation = Operation::Add;  // Add, Subtract, Multiply, Divide


## Что демонстрирует

- ✅ Enum и pattern matching
- ✅ Обработка ошибок через `Result<T, E>`
- ✅ Функции с типизацией
- ✅ Базовый синтаксис Rust

## Технологии

- Rust 2021 edition
- Cargo
